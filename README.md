# AJEX BINU — Portfolio (Version 2.0)

Hey! This is my personal portfolio site where I showcase my skills, achievements, and projects. Built with HTML, CSS, and JS, and fully responsive for mobile and desktop.  

---

## What’s Inside

- **Hero Section:** A quick intro with a cool typing effect showing what I do.  
- **About Me:** Who I am, what I do, and a CV download link.  
- **Skills:** Technical skills with animated progress bars and professional soft skills.  
- **Achievements:** Highlights of my accomplishments with icons and animations.  
- **Contact:** A simple form (via Formspree) so you can reach me easily.  
- **Smooth UX:** Scroll-to-top button, reveal animations, and mobile-friendly navigation.

---

## Tech Used

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- JavaScript (Typing effect, scroll reveal, progress bars)  
- FontAwesome icons
